## Core Philosophy
- **Monetization Mantra**: Use psych data (OCEAN/DISC/Enneagram/moral/behavior logs) to personalize funnels—e.g., high-risk traits push to slots, high-empathy to community sales. All features feed a "Revenue Nexus" in Supabase: Tracks user journeys, predicts churn, and A/B tests manipulations (e.g., flashes/delays) for max engagement.
- **Entry Points**: 3+ ways in: 1) Free retail tools (inventory/CRM hook), 2) Gamified map/exploration (fun draw), 3) Social/multiplayer (viral pull). All converge on sales/slots.
- **Data-Driven**: Every interaction logs to Supabase for "God AI" evolution—predicts "money moves" like optimal upselling times based on mood traits.
- **Predatory Edge**: Subtle intrusions (e.g., "Psych nudge: Your low resilience suggests a quick win—spin slots?") with consent pop-ups. Rewards loyalty with buffs, punishing disengagement with "cooldowns."
- **Tech Fit**: Builds on React/TS/Tailwind/Context API/Gemini/Supabase. Add Zustand for state, Vite for builds, cloud DB for sync.

## New Money-Making Features
1. **Dynamic Pricing Engine**:
   - AI adjusts product prices in real-time based on user psych profile (e.g., impulsive traits = flash sales; cautious = bundles). Ties to POS: "Haggle Assistant" uses enneagram to suggest counters (e.g., appeal to Type 3's ego: "Elite operators get this deal").
   - Revenue Boost: A/B test pricing via manipulator engine—log reactions (e.g., cart abandons) to optimize. Funnel: Post-purchase, "Unlock slots with winnings to restock smarter."
   - Entry Point: Free trial via Ghost Portal—draws customers in, profiles them during chats.

2. **Affiliate & Upsell Vortex**:
   - Multiplayer GTN expands to "Affiliate Alleys" on the map: Users recruit "operators" (friends) for commissions on their sales/slots plays. Psych match: Pair high-dominance users as mentors.
   - Money Maker: Tiered rewards—e.g., unlock "Influence Buff" (CRM tool) after 5 referrals, which predicts buyer morals for ethical upsells (or not-so-ethical nudges).
   - Draw-In: Viral shares via AI-gen images of "success timelines" (e.g., "Your path to riches—invite a pal!").

3. **Slots Integration Hub**:
   - Seamless funnel: Map zones lead to "Neon Casino District"—text/image-based slots mini-games (Gemini-gen spins with psych tweaks: Thrill-seekers get high-volatility; conservatives get steady wins).
   - Monetization: Virtual currency from sales missions convertible to real slots entries (if legal; else simulated). Log behaviors to personalize: "Your Type 8 Challenger vibe loves risks—double down?"
   - Performance Booster: Slots wins grant "Profit Multipliers" for real inventory (e.g., +10% margins on next batch). Data: Logs gambling patterns to refine CRM (e.g., "Risk-tolerant customers buy more premium stock").
